Export Schedule

You can export your schedule in iCalendar format. The resulting calendar file will include all courses from all terms that you have marked as 'Officially Enrolled' (or for which you are an instructor).

There are a few ways to import your schedule into various calendar applications:

  2. Copy the URL below into a web based calendar application (e.g. Yahoo Calendar) or desktop calendar application (e.g. Mac OS X Calendar). You can share this URL with friends, but be aware that anyone with this URL can see any changes you make to your schedule in the future.
  3. Download your schedule and upload it to your online calendar app or open it in a desktop calendar app.
